仓库设置

支持 owner/repo 或完整 GitHub 链接。其余设置都是可选增强。

界面主题
只改档案保险库自己的面板配色，不改酒馆本体主题。

创建备份

默认打包当前用户目录。无需勾选备份范围。

维护

查看仓库占用，手动回收孤儿分块，检查快照健康。

点击“刷新空间”查看当前仓库占用。

档案库

同一仓库里的所有设备档案都会列在这里。可按设备筛选。

搜索与筛选
进入档案库后才会加载备份列表。

按路径恢复

选择要恢复的路径

“合并恢复” 会把选中的文件覆盖到本地，但保留本地其它内容。

“严格覆盖” 会先删掉你选中的那些路径，再按备份内容重建。
